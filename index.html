MogileFS-PHP-Model by Yuav

Model representation of a file inside MogileFS, complete with unit tests, mapper and adapters for save, find and delete

This is a model/mapper approach for files inside MogileFS, which is a distributed storage system. The purpose of this project is to provide a clear PHP API for working with files stored in MogileFS. All properties except paths for a file is lazyloaded on request. Multiple adapters are present: * Mysql adapter, which introduce a hack for bulk operations on * Tracker adapter, which is the native socket connection to MogileFS tracker * Test adapter, for testing your application without having a test MogileFS set up Unit tests coverage is above 90% Coding style is attempting to follow Zend Framework syntax.
